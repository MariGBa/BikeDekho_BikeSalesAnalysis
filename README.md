# BikeDekho — Bike Sales Analysis (Excel Project)

![Excel Logo](https://img.shields.io/badge/Tool-Microsoft_Excel-green)

##  Project Overview
BikeDekho is a hands-on data analysis project using **Microsoft Excel** to explore bike sales trends, customer demographics, and product preferences. The goal is to generate actionable business insights and create an interactive executive dashboard for decision-making.

---

##  Project Components

| File | Description |
|------|-------------|
| `BikeDekho_Final.xlsx` | Cleaned dataset, PivotTables, insights, and dashboard with KPIs & charts |

---

##  Skills Demonstrated
- **Data Cleaning & Preparation**: Handling duplicates, missing values, and creating calculated columns  
- **Exploratory Data Analysis (EDA)**: PivotTables, COUNTIFS, SUMIFS, IF, VLOOKUP, DATE functions  
- **Data Visualization**: Bar, Column, Pie, and Line charts; interactive slicers and filters  
- **Dashboard Design**: Executive-style layout with KPIs, segmentation, and color-coded visuals  
- **Business Insights Interpretation**: Identifying customer segments, regional performance, and product preferences  

---

##  Key Insights
- Married males aged **30–45** are the most frequent bike buyers.  
- **Mountain bikes** are the most preferred product line.  
- **Western regions** recorded the highest sales volume.  
- Income and occupation strongly influence purchasing behavior.  

---

##  Dashboard & Visualizations
- KPIs: Total Buyers, Units Sold, Average Income, Customer Count  
- Interactive PivotCharts: Sales by Region, Purchases by Gender, Age Group segmentation  
- Slicers and filters for real-time dashboard exploration  
- Clean, executive-style layout for decision-making  

---

##  How to Use
1. Open `BikeDekho_Final.xlsx` in Microsoft Excel.  
2. Navigate through sheets:
   - **Cleaned Data** — raw cleaned dataset  
   - **EDA** — PivotTables for multi-dimensional analysis  
   - **Insights** — summary of key findings  
   - **Dashboard** — interactive charts and KPIs  
3. Use **Slicers** to filter by Gender, Region, Marital Status, or Age Group.
